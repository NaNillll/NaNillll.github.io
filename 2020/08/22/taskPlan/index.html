Concept2Robot: Learning Manipulation Concepts from Instructions and Human Demonstrations

Overview

384维的vector输入policy network后输出1×7的goal g和T×7的 forces f。T为trajectory的time step，用来生成motion trajectory

Details

训练时，BERT固定，只训练最后的MLP

motion trajectory by a second-order dynamical system

生成f和g时使用了反卷积。The final task embedding from the semantic context network with shape 1 × 384 is simultaneously fed into an MLP (equivalent to 1D upconvolution) to generate the goal g with shape 1 × 7 and into a separate 1D upconvolution network to generate the forces f with shape T ×7, where T is the number of timesteps in the trajectory.

video classifier that is trained to classify actions in Sth Sth videos. For a given task Ti and a video V, the classification score of the corresponding category is used as a reward signal R(V, Ti) to update the policy network

Single-Task Learning

Given the video classifier as a substitute for the reward, we frame this problem as a reinforcement learning problem per task Ti

To solve the Contextual bandit problem, we combine Deep Deterministic Policy Gradients (DDPG) with the Cross Entropy Method (CEM) .The DDPG critic network is used to approximate the Q value function. The output of the DDPG actor network is used to initialize CEM. We run CEM for four iterations to select actions in both the training and test stages.

In the training stages, we store the transition (s, a, r) where s represents the state, a is the action selected after CEM, r is the reward from the video classifier. The critic loss is Lc = ||r− Q(s, a)||^2 , and the actor loss is La = −Q(s, Actor(s)), where Actor(s) is the output of the actor network.

Multi-Task Learning

After training single-task policies, we now have expert models for each task that we can use to train a multi-task model through imitation learning

For each task Ti , we use the corresponding single-task policy πi to produce N rollouts of state-action tuples (s,a,y) and save them into a large replay buffer. s is e feature vector, a is motion trajectory parameter, y is actual end effector trajectory
