Texuring Defination

使用纹理来修改物体的表面的每一个位置，纹理可以来源于

  1. 图片
  2. 函数
  3. 其他数据结构

主要在shading中进行修改

Texture Pipeline

The projector Function

将三维坐标投影到二维

The Corresponder Function

映射函数，

但是可能会超出该范围，需要对应进行填充

Texture Magnification

处理纹理太小，分辨率不够的情况

主要通过插值

Bilinear Interpolation

Texture Magnification

纹理太大，分辨率太高

其实这和反采样是很相似的问题

Supersampling

太慢

Mipmap

提前计算出纹理降低分辨率后的结果，在需要取纹理时找到对应位置并插值即可

为了更好的过度，比如为第1.3层插值，需要用到三线性插值——在不同层得到的结果间插值

Anisotropic Filtering

在各个方向上的取值频率不同

同时取样形状也不一定是矩形

Bump Mapping凹凸贴图

修改表面法线的方向。可能只是在shading部分使用该法线，而在其他部分使用顶点本身的法线

heightfield

一种方法是存储顶点的高度，并计算法线

normal map

一种方法是将法线直接存储在贴图中

然而，法线显然无法存储在世界坐标系下，一般存储在切线坐标系下：

每个顶点的切线坐标系下存储在每个顶点中

Parallax Mapping视差贴图

凹凸贴图的一个问题是这种凹凸性本身是有问题的——既不会因为凹凸产生相互的遮挡，在视角移动后凹凸效果也不会变化

视差贴图本身是通过贴图，预测某个视角下应该看到物体的哪个位置，通过偏移进行取样

https://blog.csdn.net/HappyKocola/article/details/72812081

Environment Map

最常见的是球形贴图

用于反射，透射，天空盒等
