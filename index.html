Bounding Volumes

常用的加速方法是包围盒

常用的是Axis-Aligned Bounding Box (AABB)，AABB式包围盒。这样的包围盒在计算上有优势

相当于是三组对称的平面

判断是否相交，需要计算所有进入平面组中的最大值和所有离开平面组的最小值，如果前者小于后者，说明完全进入包围盒时还没有离开包围盒，则说明ray与包围盒相交

但是ray是射线而不是直线，因此还需要检查与离开包围盒是否t>0

综上需要满足两个条件

Uniform Spatial Partitions (Grid)

一种划分包围盒的方法是空间内均匀划分

在遍历时，只需要依次检查光线与当前分割盒周围的盒是否相交

然后依次检查盒中物体与ray是否相交

盒子不能过多：遍历时检查次数太多

盒子不能过少：只有一个盒子相当于没有加速

一般三维情况下选择27*物体数量

这种方法适用于物体在空间内均匀分布的情况，而不适用“体育馆中的茶壶”

Spatial Partitions

KD-Tree

沿x，y，z轴划分，只有叶子节点内有物体

Bounding Volume Hierarchy (BVH)

  1. 找到包围盒
  2. 分割盒内物体。尽量沿最长的边平均分割
  3. 形成新的包围盒；只在叶子节点内存储物体
  4. 直到每个叶子节点内物体都不多

漫反射强弱与光照强度有关，而光照强度除了和光照本身属性相关以外，还与与法线夹角和到光源距离有关

Specular Term (BlinnPhong)

计算反射的高光项，这一项与视角有关。显然视角越靠近反射方向，高光应该越明显

但是计算这个反射方向太花时间了，因此通过计算半程向量与法线的夹角，近似度量这一项

Ambient Term

环境光照，完全近似的结果，用来弥补被忽略的各种光

Sampling采样

一般通过离散取值来进行采样

一般使用三角面片作为Mesh的Primitive

  1. 最基本的多边形
  2. 容易确定是否在内部
  3. 一定是在一个平面内
  4. 容易插值

判断是否在三角形内可以使用叉积

叉积具有方向性，反映旋转的方向。依次计算Qp0和Qp1，Qp1和Qp2，Qp2和Qp3的叉积，如果都同号说明在三角形内部，否则外部

在采样时，可以使用逐行扫描的方法加速

同时可以使用包围盒，更快的判断是否在三角形内部

Aliasing走样

直接采样容易发生走样Aliasing，主要是因为采样频率太低，信号变化过快

这种错误又被称为Sampling Artifacts(Errors/Mistakes/Inaccuracies)

一种常见的走样是因为时间频率过快，如倒转的轮子

一种常见的走样是空间上的，颜色值变化过快，比如锯齿和摩尔纹

Antialiasing反走样

先滤波后采样

滤波除去高频部分，模糊后采样

Supersampling，MSAA

将一个像素扩展成许多个像素，依次计算子像素的属性，并插值得到最终的结果

除此以外还有许多抗锯齿方式

另一种方法是使用快慢指针，fast一次前进两个位置，slow一次。假如有环，那么fast一定先进入环。当slow进入环后，slow在一圈之内一定会被fast追赶上

假设在紫色位置两者相遇，则,

假如此时slow继续前进a，会到达相交点。因此，此时令新指针从起始处运动，则会与slow交于目标位置

19. 删除链表的倒数第N个节点

只要找到第N个节点就可以解决问题了。令快指针先前进N，然后慢指针与他一起前进直到快指针到达结尾NULL，此时的位置就是倒数第N个节点

至于删除时，可能删除的恰好是头结点，因此需要单独讨论，或者加上dummy node
