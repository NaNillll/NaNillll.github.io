Accelerating Ray-Surface Intersection

加速检测ray与物体的碰撞检测

Bounding Volumes

常用的加速方法是包围盒

常用的是Axis-Aligned Bounding Box (AABB)，AABB式包围盒。这样的包围盒在计算上有优势

相当于是三组对称的平面

判断是否相交，需要计算所有进入平面组中的最大值和所有离开平面组的最小值，如果前者小于后者，说明完全进入包围盒时还没有离开包围盒，则说明ray与包围盒相交

但是ray是射线而不是直线，因此还需要检查与离开包围盒是否t>0

综上需要满足两个条件

Uniform Spatial Partitions (Grid)

一种划分包围盒的方法是空间内均匀划分

在遍历时，只需要依次检查光线与当前分割盒周围的盒是否相交

然后依次检查盒中物体与ray是否相交

盒子不能过多：遍历时检查次数太多

盒子不能过少：只有一个盒子相当于没有加速

一般三维情况下选择27*物体数量

这种方法适用于物体在空间内均匀分布的情况，而不适用“体育馆中的茶壶”

Spatial Partitions

KD-Tree

沿x，y，z轴划分，只有叶子节点内有物体

Bounding Volume Hierarchy (BVH)

  1. 找到包围盒
  2. 分割盒内物体。尽量沿最长的边平均分割
  3. 形成新的包围盒；只在叶子节点内存储物体
  4. 直到每个叶子节点内物体都不多

Specular Term (BlinnPhong)

计算反射的高光项，这一项与视角有关。显然视角越靠近反射方向，高光应该越明显

Sampling采样

一般通过离散取值来进行采样

一般使用三角面片作为Mesh的Primitive

  1. 最基本的多边形
  2. 容易确定是否在内部
  3. 一定是在一个平面内
  4. 容易插值

判断是否在三角形内可以使用叉积

叉积具有方向性，反映旋转的方向。依次计算Qp0和Qp1，Qp1和Qp2，Qp2和Qp3的叉积，如果都同号说明在三角形内部，否则外部

在采样时，可以使用逐行扫描的方法加速

同时可以使用包围盒，更快的判断是否在三角形内部

Aliasing走样

直接采样容易发生走样Aliasing，主要是因为采样频率太低，信号变化过快

这种错误又被称为Sampling Artifacts(Errors/Mistakes/Inaccuracies)

一种常见的走样是因为时间频率过快，如倒转的轮子

一种常见的走样是空间上的，颜色值变化过快，比如锯齿和摩尔纹

Antialiasing反走样

先滤波后采样

滤波除去高频部分，模糊后采样

Supersampling，MSAA

将一个像素扩展成许多个像素，依次计算子像素的属性，并插值得到最终的结果

除此以外还有许多抗锯齿方式
